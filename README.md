# 赤木ゼミ website

## 使用技術

- [deno](https://deno.land/)
- [fresh](https://fresh.deno.dev/)
### CMS
- [NotionAPI](https://developers.notion.com/)
### デプロイ先
- [Deno Deploy](https://deno.com/deploy)

## Usage

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
